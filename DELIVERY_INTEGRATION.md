# Интеграция расчета стоимости доставки

## Что сделано:

1. ✅ Добавлен расчет стоимости доставки через Яндекс Доставку при создании заказа
2. ✅ Стоимость доставки автоматически добавляется к итоговой сумме заказа
3. ✅ Настроен адрес отправления: "Москва, 1-й проезд Марьиной Рощи, дом 7/9"

## Как это работает:

### При создании заказа:

1. Если `delivery_method == "delivery"` и указан `customer_address`:
   - Система рассчитывает стоимость доставки через Яндекс Доставку API
   - Использует адрес отправления из настроек (1-й проезд Марьиной Рощи, дом 7/9)
   - Использует адрес клиента как адрес назначения
   - Добавляет стоимость доставки к `total_amount` заказа

2. Если `delivery_method == "pickup"`:
   - Стоимость доставки = 0
   - Заказ создается без расчета доставки

### Адрес отправления:

Настроен в `app/config.py`:
- **Адрес**: "Москва, 1-й проезд Марьиной Рощи, дом 7/9"
- **Координаты**: [37.6000, 55.8000] (примерные, можно уточнить через геокодер)

### Геокодирование адреса клиента:

- Если настроен `YANDEX_GEOCODER_API_KEY`, система автоматически преобразует адрес клиента в координаты
- Если геокодер не настроен, используются координаты по умолчанию (центр Москвы)

## API изменения:

### CreateOrderRequest:

Добавлено поле:
```python
delivery_method: str = "pickup"  # "pickup" или "delivery"
```

### Пример запроса:

```json
{
    "customer_name": "Иван",
    "customer_phone": "+79991234567",
    "customer_address": "Москва, Тверская улица, 10",
    "items": [
        {
            "product_id": "uuid",
            "quantity": 1
        }
    ],
    "payment_method": "cash",
    "delivery_method": "delivery"  // Новое поле
}
```

## Настройка:

### В `.env`:

```bash
# Яндекс Доставка (обязательно для расчета доставки)
YANDEX_DELIVERY_TOKEN=ваш_токен

# Яндекс Геокодер (опционально, для точного геокодирования адресов)
YANDEX_GEOCODER_API_KEY=ваш_ключ_геокодера
```

### Адрес отправления:

Настроен в `app/config.py`:
- `pickup_address_fullname`
- `pickup_address_coordinates`
- `pickup_address_city`
- `pickup_address_country`
- `pickup_address_street`

## Важно:

⚠️ **Расчет доставки происходит автоматически при создании заказа, если:**
- `delivery_method == "delivery"`
- Указан `customer_address`

⚠️ **Если расчет доставки не удался:**
- Заказ все равно создается
- Стоимость доставки = 0
- Ошибка логируется, но не прерывает создание заказа
